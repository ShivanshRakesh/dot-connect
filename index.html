  • DOT CONNECT

Are you sure you want to RESET?

Connect the required number of dots in a straight line oriented in any direction to win. The dots drop in a vertical column and stack one over another. The player who does this first, wins the game. The number of dots required to win can be selected from the options menu () in the navigation bar. The number of players can be between 2-4. Also, each player gets exactly one chance for UNDO. Consecutive UNDOs is not allowed.
